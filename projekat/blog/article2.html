Relevance of Linux in Cyber Security


It's no secret that various Linux distributions are used by hackers worldwide, but what's the appeal? Why not simply use Windows or some other operating system? Furthermore, which distributions are designed for the purpose of penetration testing? Let's take a look at the table below.


Windows Linux
User management Originally written for one user-one computer Multi-users with various access rights
Malware Since 2009, over 9 million malwares have targeted Windows. As of 2006, only 800 malwares had been discovered.
Open vs Closed Claims its platform is more secure because of a comprehensive approach using the Security Development Lifecycle Claims its platform is more secure because all its code is reviewed by lots of people, therefore more bugs are detected (also known as the Linux Law).
Threat detection and solution Threat detection and solution is relatively fast. Releasing a patch with a fix for a major threat can take up to 3 months.


As you can see, the obvious choice is Linux.

In the words of Eric S. Raymond, a world-famous hacker and open-source advocate:

Yes, there are other operating systems in the world besides Unix. But they're distributed in binary â€” you can't read the code, and you can't modify it. Trying to learn to hack on a Microsoft Windows machine or under any other closed-source system is like trying to learn to dance while wearing a body cast.
Furthermore,
Unix is the operating system of the Internet. While you can learn to use the Internet without knowing Unix, you can't be an Internet hacker without understanding Unix. For this reason, the hacker culture today is pretty strongly Unix-centered.
Linux is open-source and free, which means that anyone can use it for any type of work, edit, copy, distribute various aspects of a Linux based OS without violating any copyright law, term or condition. Let's get into the various distributions used by Cyber Security professionals.


Distributions designed for pentesting

Kali Linux
Armed with over 600 pre-installed pentesting programs, from Armitage, Nmap, Wiresshark, John the Ripper, Aircrack to Burpsuite and many more, Kali Linux is the AK-47 of Linux distros. It gained popularity among newbies because it's relatively easy to use.

Parrot
Used mostly by penetration testers who need a cloud-friendly environment with online anonymity and an encrypted system, Parrot OS is a Debian based, security OS.
